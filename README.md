Kicer's C Library
===============

**INSTALL**

- `$ make` to make static library

- `$ USE_DYNAMIC_LINK=1 make` to make dynamic library

	> Run dynamic link demo: `$ LD_LIBRARY_PATH=./lib ./demo/k-test`

- `$ make install` for install

- `$ make uninstall` for uninstall

- `$ make clean` for clean
